# Predictive Ergonomic Risk Modeling using Machine Learning

This project aims to develop a machine learning framework for predicting ergonomic risk from workplace assessment data.

## Research Motivation

Work-related musculoskeletal disorders (WMSDs) remain one of the leading causes of occupational injuries worldwide. Predictive AI models can support proactive ergonomic interventions and contribute to future human-robot collaboration in occupational environments.
